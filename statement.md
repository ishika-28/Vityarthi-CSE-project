## Problem Statement
I notice many people have trouble keeping track of the spending and seeing the patterns. The trouble leads to the budgeting problems. Missed chances to save. I have seen this happen so many times, the Expense Tracker fixes the problem. The Expense Tracker gives a way to record the expenses. The Expense Tracker lets the user sort the expenses into the categories. The Expense Tracker lets the user view the expenses to see the patterns. When the user opens the Expense Tracker the user can add an expense in seconds. Pick the category from a simple list. The Expense Tracker keeps everything organized.
## Scope of the Project
I am building a command line Expense Tracker for my use. The Expense Tracker uses Python, Pandas and Matplotlib. The scope of the Expense Tracker includes:
1. Inputting expense records with date, category, and amount.
2. Categorizing expenses for organized tracking.
3. Summarizing expenses to show total, average, maximum, and minimum values.
4. Visualizing expense data through graphical reports (bar charts).
5. Limitation: I built the application for use. The application only tracks sessions. The application does not use a database.

## High Level Features
1. Expense Entry: Add expenses with validation for proper numeric input.
2. Expense Categorization: I use Expense Categorization because Expense Categorization helps me sort each expense. Expense Categorization lets me put an expense into a category such, as food, travel or shopping. Expense Categorization shows where the money goes. I rely on Expense Categorization to keep track of my spending.
3. View Expenses: I open View Expenses. I see all the recorded expenses, in the table.
4. Summary Generation: Compute and show total, average, minimum, and maximum expense statistics.
5. Visualization: Generate bar charts to show the expenses over time using Matplotlib.
6. User-friendly Menu: Simple command-line menu for ease of use.
7. Error Handling: Provides feedback for invalid inputs and empty records.
